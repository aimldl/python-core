### [3.1.1. 数](https://docs.python.org/ja/3.14/tutorial/introduction.html#numbers)

#### Let's summarize the 用語 before we start. 
| 用語 | ふりがな | 意味 |
|---|---|---|
| 算術 | さんじゅつ | 数の計算 |
| 演算 | えんざん | 計算の操作 |
| 演算子 | えんざんし | + や − などの記号 |
| 被演算子 | ひえんざんし | 演算の対象となる数や式 |

#### Tip
* 変数 _: 対話モードでは、最後に表示された結果は変数 _ に代入されます。

#### 式の文法

式の文法は素直なものです: 
* 演算子 （えんざんし） + 、 - 、 * 、 / によって算術演算（さんじゅつえんざん）を行うことができ、
* 丸括弧 (()) をグループ化に使うことができます。 

#### 例えば:
```
2 + 2
50 - 5*6
(50 - 5*6) / 4
8 / 5  # 除算は常に浮動小数点数(float)を返す
```

* 数値型
  * 整数 (例えば、 2 、 4 、 20) は int 型であり、
  * 小数部を持つ数 (例えば、 5.0 、 1.6) は float 型です。
  * 数値型については後のチュートリアルでさらに見ていきます。

#### / vs. //
* 除算 (/) は常に浮動小数点数を返します。 
* // 演算子は 整数除算 を行い、整数値を返します; 
* 剰余は、% で求めます。

```
17 / 3  # 除算は浮動小数点数を返す
17 // 3  # 整数除算は小数部を破棄する
17 % 3  # % 演算子は剰余を返す
5 * 3 + 2  # 切り捨てられた商 * 除数 + 剰余
```
#### ** 演算子
Python では、冪乗を計算するのに ** 演算子が使えます

```
5 ** 2  # 5の2乗
2 ** 7  # 2の7乗
```

#### 変数と等号
等号 (=) は変数に値を代入するときに使います。代入を行っても、結果は出力されず、次の入力プロンプトが表示されます。:

```
width = 20
height = 5 * 9
width * height
```

```
print( width ) 
print( height )
```

変数が "定義" されていない (つまり値が代入されていない) 場合、その変数を使おうとするとエラーが発生します:
```
n  # 未定義の変数にアクセスする
```

#### 
浮動小数点を完全にサポートしています。演算対象の値(オペランド)に複数の型が入り混じっている場合、演算子は整数のオペランドを浮動小数点型に変換します:

```
4 * 3.75 - 1
```

#### 変数 _
対話モードでは、最後に表示された結果は変数 _ に代入されます。このことを利用すると、Python を電卓として使うときに、計算を連続して行う作業が多少楽になります。以下に例を示します:

```
tax = 12.5 / 100
price = 100.50
price * tax

_ + price

round(_, 2)
```

* この変数には読取りだけを行い、明示的な代入を行ってはいけません 
* --- そんなことをすれば、同じ名前で別のローカル変数が生成され、元の特別な動作をする組み込み変数を覆い隠してしておかしなことになってしまうかもしれません。

#### Decimal, Fraction, 複素数 (j もしくは J)
int と float に加え、 Python は Decimal や Fraction などの他の数値型もサポートしています。 複素数 も組み込み型としてサポートしており、 j もしくは J 接尾辞を使って虚部を示します (例: 3+5j)。
