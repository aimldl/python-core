# [3.形式ばらない Python の紹介](https://docs.python.org/ja/3.14/tutorial/introduction.html)
このマニュアルにあるサンプルの多くは、対話プロンプトで入力されるものでもコメントを含んでいます。 

いくつかの例です:

```
# これは最初のコメント
spam = 1  # そしてこれは2つめのコメント
          # ...そして3つめ!
text = "# これはクォーテーションの間なのでコメントではない。"
```
* Python におけるコメント文は、ハッシュ文字 # で始まり、物理行の終わりまで続きます。
* Pythonはコメントを解釈しません。なので、サンプルコードを実際に入力して試して見るときは、コメントを省いても大丈夫です。
